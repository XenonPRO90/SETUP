## 🐛 Если что-то не работает:

### Проблема: npm install падает с ошибкой
**Решение:** Выполните шаг 1 (исправление прав npm)

### Проблема: Порт 3000 занят
**Решение:** 
```bash
lsof -ti:3000 | xargs kill -9
```
Или используйте другой порт:
```bash
npm run dev -- -p 3001
```

### Проблема: Ошибки TypeScript
**Решение:** Проверьте что все файлы на месте:
```bash
ls -la app components
```

### Проблема: Стили не применяются
**Решение:** Перезапустите dev-сервер (Ctrl+C, потом npm run dev)
