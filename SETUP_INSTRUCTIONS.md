## Важно! Проблема с npm

У вас есть проблема с правами доступа к npm cache. Перед установкой зависимостей нужно исправить:

```bash
sudo chown -R 501:20 "/Users/romanpisulov/.npm"
```

Введите пароль администратора когда попросит.

## После исправления прав:

### 1. Установить зависимости

```bash
cd /Users/romanpisulov/Desktop/setup-website
npm install
```

### 2. Запустить dev-сервер

```bash
npm run dev
```

### 3. Открыть в браузере

Откройте [http://localhost:3000](http://localhost:3000)
